# Prompt para Figma Make — Site Responsivo “Verso Livre”

Crie um **site responsivo** para a plataforma **Verso Livre**.

A plataforma apoia escolas na criação de eletivas de escrita criativa com foco em diversidade, pertencimento e protagonismo LGBTQIAPN+, oferecendo materiais de apoio, acompanhamento dos estudantes e uma biblioteca digital acessível para publicação das histórias.

O site deve ser pensado para **desktop**, **tablet** e **celular**, sem quebrar em nenhuma resolução. No desktop, deve parecer um site web completo. No celular, deve se adaptar visualmente para uma experiência parecida com a referência enviada: jovem, colorida, com cards grandes, bordas arredondadas, fundo claro e navegação simples.

---

## 1. Identidade visual obrigatória

Use a referência visual enviada como base de estilo.

A interface deve ter:

- fundo claro/off-white;
- cards grandes e arredondados;
- cores vibrantes;
- visual jovem, educacional e criativo;
- sensação de aplicativo moderno, mas aplicado em formato de site;
- bastante espaçamento;
- sombras suaves;
- elementos de livros, escrita, histórias e leitura;
- layout limpo;
- textos curtos e diretos.

---

## 2. Cores principais

Use obrigatoriamente estas cores:

- Laranja vibrante: `#FB5603`
- Azul/roxo escuro: `#1B0C73`
- Rosa magenta: `#D81DA0`

Cores auxiliares:

- Fundo claro: `#FFF8F6`
- Fundo alternativo: `#FAF4F2`
- Texto principal: `#111111`
- Texto secundário: `#4B4B4B`
- Branco: `#FFFFFF`

---

## 3. Tipografia

Use **Poppins** como fonte principal.

Hierarquia:

- Títulos: Poppins Bold;
- Subtítulos: Poppins SemiBold;
- Textos: Poppins Regular;
- Botões: Poppins SemiBold;
- Cards: Poppins Bold para títulos e Poppins Regular para descrições.

---

## 4. Estilo visual inspirado na referência

O visual deve lembrar a tela enviada:

- logo colorida e jovem;
- cards quadrados ou retangulares com cantos arredondados;
- card laranja para ações principais;
- card rosa para criação de história;
- card azul/roxo escuro para conteúdos de leitura;
- livros com gradiente laranja, rosa e roxo;
- fundo claro;
- títulos pretos;
- botões grandes;
- ícones simples;
- sombras suaves nos cards;
- seções bem separadas.

Não criar moldura de celular no desktop.  
O site deve ser web, mas quando aberto no celular deve ficar parecido com a referência.

## 26. Regras de responsividade obrigatórias

O site não pode quebrar em celular.

### Desktop

- largura máxima do conteúdo: 1200px a 1280px;
- hero em duas colunas;
- grids com 3 ou 4 colunas;
- dashboards com sidebar lateral;
- tabelas completas.

### Tablet

- hero pode virar duas colunas menores ou uma coluna;
- grids com 2 colunas;
- sidebar pode recolher;
- espaçamento reduzido.

### Mobile

- header com menu hambúrguer;
- logo centralizada;
- cards em uma coluna ou duas colunas quando couber;
- botões com largura total;
- tabelas viram cards;
- editor em coluna única;
- carrossel horizontal para livros;
- cards principais **Minhas histórias** e **Criar história** podem ficar lado a lado;
- nenhum texto deve vazar para fora da tela;
- nenhum card deve cortar conteúdo;
- usar padding lateral de 20px a 24px;
- manter fundo claro;
- manter aparência parecida com a referência enviada.

- ranking;
- rede social aberta;
- feed público com comentários livres;
- chat em tempo real;
- loja;
- pagamento;
- gamificação competitiva;
- IA escrevendo a história pelo aluno.

- autoria;
- professor como agente;
- segurança;
- acessibilidade;
- publicação;
- baixa conectividade;
